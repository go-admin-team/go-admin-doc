# 验证

检查一下Golang版本信息，

$ go version
go version go1.16.4 darwin/amd64
1
2

看到以上信息说明当前工作环境已经安装成功了go1.16.4 darwin/amd64，当前笔者因为是 MacOS 所以提示的是darwin。这个信息会根据操作系统当前安装的版本提示。windows和linux 会有所不同；
